What is HTML?

HTML Stands for Hypertext Markup Language. It's a Markup language that web developer use to structure and describe the content of web page (not a programming language).

HTML cosists of elements that describe different tpye of contents: paragraph , heading , links , images , video etc. web browsers understand HTML and render html code as website.

In HTML each elements is made of 3 parts:

  1. The opening tags
  2. The closeing tags
  3. Actual elements
